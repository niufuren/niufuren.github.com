Research Interests

  • Query Processing. The main focus of my thesis is on adaptive join algorithms for streamed and semi-streamed data. We have developed DINER, an algorithm for joining two finite streams with large applicability in data integration and online aggregation. Our publication describing this algorithm received the best student paper award in ICDE 2009. Next we developed MINER, a multi-way join operator that extends the main ideas of DINER to multiple inputs.
    SSIJ is a Semi-Streamed Index Join algorithm with applicability in active data warehouses or DSMS systems where the join between a continuous stream and a local, on-disk relation represents a fundamental operation.

  • Transaction Processing on Replicated Databases. During my internship at Microsoft Research I was working on a novel transaction isolation level suitable for replicated database systems. We developed Serializable Generalized Snapshot Isolation, a concurrency control algorithm that ensures serializable transaction execution in replicated databases.

  • Physical Design Tuning. I am currently studying techniques for automatic index management in a database system. We are investigating the effect of introducing an extra option in a database systems, which is not to propagate updates to certain indices based on their contribution to the execution of the workload statements.
